<h2 style="text-align: center;">Use FreeRTOS with STM32F103C8T6 microcontroller to read BH1750 value and display it on NOKIA5110 screen with 2 buttons</h2>

## NOKIA5110

- **PA0 = Light**
- **PA1 = DC**    
- **PA2 = CE**
- **PA3 = DIN**            
- **PA4 = RST**    
- **PA5 = CLK**
- **VCC = 5V**
- **GND = GND**

## BH1750

- **PB6 = SCL**
- **PB7 = SDA**
- **VCC = 5V**
- **GND = GND**

## Button

- **PA6 = Button 1 Pin 1**
- **PA7 = Button 2 Pin 1**
- **GND = Button 1 Pin 2**
- **GND = Button 2 Pin 2**
